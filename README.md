Calorie Calculator is a project that aims to create a mobile application. It is the most complex project I have ever worked on. Making and testing it was a real pleasure for me. A total of three people worked on it, and this is not its final version. It will be finalized as soon as possible and will be even more complex and fully completed. So far, it has the following functionalities: 
1.  Development of a mobile application for Android with .NET MAUI, which 
provides a convenient interface for tracking calorie intake. 
2.  Building a server architecture with ASP.NET Core Web API, providing 
secure and scalable communication between the client and the database. 
3.  Using a PostgreSQL database, hosted in the cloud via Aiven, to 
store user data, food products and calorie 
plans. 
4.  Implementation of modern architectural patterns such as Repository Pattern, 
CQRS with MediatR and layered architecture.
Using four hardware features of the mobile device: a camera for scanning barcodes (ZXing.Net.Maui), a camera for capturing a profile picture/selfie (MAUI MediaPicker), a biometric fingerprint sensor as two-factor authentication (Plugin.Fingerprint), and Wi-Fi/mobile data for network communication with the API server. 
5.  Enabling teamwork through GitLab with a GitFlow version management strategy.

If you want to learn more about Calorie Calculator or see what the demo of the application actually looks like - you can do so in the documentation and presentation attached to the project, which I have shared in this repository.
